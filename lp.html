Features

3つの特徴

指示を出す場所はDiscord。あとはCortexが考えて、動いて、報告する。

⚡

並列サブエージェント

タスクを自動で分解し、依存関係を分析。独立したタスクはClaude Codeサブエージェントで同時並列実行。直列の3倍速でプロジェクトが進む。

📈

リアルタイム進捗管理

git commitにタグを付けるだけで、Supabaseのマイルストーンが自動更新。Cortex Viewダッシュボードで全プロジェクトの進捗を一覧できる。

🧠

コーチング&ブロッカー検知

3日以上放置されたブロッカーを自動検知して通知。セッション開始時にプロアクティブに声かけし、プロジェクトの停滞を防ぐ。

How it works

Cortexが動く8ステップ

Discordで一言指示するだけで、裏側では8つのステップが自動で走ります。

1

Discord でメッセージ受信

ユーザーが @Cortex ○○して とDiscordで指示。MCPプラグインがメッセージをClaude Codeに中継。

2

モード判定

メッセージ内容を分析し、Mission（実行）/ Status（確認）/ Chat / Coaching の4モードに自動振り分け。

3

コンテキスト取得

Supabaseからプロジェクト一覧・マイルストーン・ブロッカーを取得し、現在の全体像を把握。

4

タスク分解

ユーザーの指示を独立したサブタスクに分解。依存関係を分析してフェーズ分けする。

5

並列エージェント起動

依存関係のないタスクをClaude Codeサブエージェントで同時起動。各エージェントが独立してコードを書く。

6

Git コミット & 自動進捗更新

コミットメッセージの [Project:Milestone] タグをpost-commitフックが検知し、Supabaseのマイルストーンを自動で完了にマーク。

7

ダッシュボード自動更新

Cortex ViewがSupabaseのリアルタイムサブスクリプションで即座に反映。ブラウザを開いていれば自動で最新状態に。

8

Discord に結果報告

全タスク完了後、実行結果のサマリーとダッシュボードリンクをDiscordに自動送信。
